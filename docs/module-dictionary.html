<inner> modifyKey(anObj, keyName, newValue)

修改object的key对应的值

Parameters:
Name Type Description
anObj object

要修改内容的对象
keyName string

需要修改的键名
newValue value

需要设置的新值
Source:
schemas/dictionary.js
Returns:

返回修改了键名对应值的新对象

Type
object
Example
var anObj = {
        "a": 1,
        "b": false,
        "c": "hello"
    };
    var newObj = modifyKey(anObj, "b", true);
    console.log(newObj);

    // 修改了anObj.b的值，并返回新对象
